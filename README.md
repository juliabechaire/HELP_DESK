# 🛠️ Help Desk - Support Management System

This project was developed as a practical assignment during the **second semester** of the **Computer Engineering** program for the **Data Structures** course. Its main objective was to apply theoretical concepts of data structures by developing a **Help Desk support management system** using the **C programming language**.

---

## 🚀 Data Structures Concepts Applied

To efficiently organize, prioritize, and manage support requests, the following data structures were implemented:

- **Dynamic Queues:** Used to manage the order of incoming support requests following the **FIFO (First-In, First-Out)** principle.
- **Linked Lists:** Used to store and manage completed support tickets and user records, allowing dynamic insertion and removal without memory waste.
- **Dynamic Memory Allocation:** Strategic use of `malloc` and `free` to allocate memory only when needed during runtime, preventing memory leaks.
- **Pointers and Structures:** Fundamental for implementing linked nodes that efficiently connect and manage the system's data.

---

## 💻 System Features

- Create and register new support tickets with user information, problem description, and category.
- Manage the support queue following the correct request order.
- View and manage the history of completed support tickets.
